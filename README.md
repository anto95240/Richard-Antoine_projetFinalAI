# 🕰️ ChronoVoyage - Agence de Voyage Temporel

Webapp interactive pour une agence de voyage temporel fictive, créée avec l'aide de l'IA générative.

## 🛠️ Stack Technique
- React + TypeScript
- Tailwind CSS + shadcn-ui
- Framer Motion (animations)
- Vite

## ✨ Features
- Landing page interactive (Hero, Fonctionnement, Témoignages)
- Catalogue de 3 destinations temporelles (Paris 1889, Crétacé, Florence 1504)
- ChronoBot : Chatbot interactif flottant avec réponses rapides

## 🤖 IA Utilisées
- Code & UI : Lovable (Génération assistée par IA)
- Visuels : Gemini (Projet 1)

```bash
